Multiroute

Multiroute is a jquery plugins to add a google maps with direction. It give you Multiple Routes from a position to your destination. Easily access multiple route of Google Maps API and it is show you the Direction Panel of every Route. So you can avoid the traffic jam.

Features

  1. Show Multiple Route for direction
  2. Crossbrowser
  3. Show text direction
  4. Autodetect Location
  5. Show Traffic Layer
  6. Multiple points/waypoints with multiple routes (new)

You can use the index.html as the sample

The jQuery Plugin file is:

  1. js/gmaps.multiroute.js
  2. css/multiroute.css
